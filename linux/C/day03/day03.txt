在某些场景下必须使用存储区的地址
    表示存储区
变量名称前加&可以获得变量对应存储区
    的地址
可以采用%p作为占位符把地址数据打印
    在屏幕上

变量可以用来代表一组相关数字
这些数字在任何时候只使用其中一个，
    它们的处理方式必须类似

字符类型里包含256个不同的整数，每个
    整数可以代表一个字符（例如'a',
	'&'等）
整数和字符是同一个东西的两种不同
    表示方式
ASCII码表里记录了所有字符和整数
    之间的对应关系
'a'      97
'A'      65
'0'      48 

ASCII码表里所有小写英文字母是连续
    排列的，'a'对应的整数最小，'z'
	对应的整数最大
ASCII码表里所有大写英文字母之间也有
     同样的规律
ASCII码表里所有阿拉伯数字字符之间
    也有同样的规律

'd' - 'a' 等于  'D' - 'A'
'd' - 'a' 等于  '3' - '0' 等于 3 - 0

所有字符数据被分成两组，每组里包含
    128个不同的字符
其中一组字符数据两种表示方式之间
    的对应关系在所有计算机上都一样。
	它们的整数范围从0到127
另外一组字符数据两种表示方式之间
    的对应关系在不同计算机上可能
	不同，它们的整数范围可能从-128
	到-1也可能从128到255

无符号字符类型也是一种数据类型
无符号字符类型的名称是unsigned char
这个数据类型里同样包含256个字符数据，
    这个类型里所有字符数据对应的整数
	在所有计算机上都一样，范围是
	从0到255

'\n'      换行字符
'\r'      回车字符
'\\'      代表字符\
'\''      代表字符'
'\"'      代表字符"

短整数也是一种数据类型
它的名称是short
这个类型里包含65536个不同的整数，
    其中一半是负数，另外一半是
	非负数。这些数字以0为中心
	向两边扩展
无符号短整数类型名称是unsigned short
这个类型里同样包含65536个整数，他们
    从0开始向正数方向扩展

长整数类型名称是long
它里面包含2的32次方个不同的整数，
    其中一半是负数，另外一半是非
	负数，以0为中心向两边扩展
无符号长整数类型名称是unsigned long
它里面同样包含2的32次方个不同的整数，
    这些整数从0开始向正数方向扩展

整数类型名称是int
在我们的计算机上整数类型和长整数
    类型一样
无符号整数类型名称是unsigned int
在我们的计算机上它和无符号长整数
     类型一样

所有只包含整数的类型数字范围重叠
    并且逐渐扩大

程序中不带小数点的数字默认是整数
    类型

可以在不带小数点的数字后加u表示这个
    数字的类型是无符号整数类型

C语言中采用浮点类型表示带小数点
    的数字
浮点类型分成单精度浮点类型和双精度
    浮点类型
双精度浮点类型可以记录更多小数点
   后面的数位
单精度浮点类型名称是float
双精度浮点类型名称是double

程序中带小数点的数字默认为双精度
    浮点类型
可以在带小数点数字后加f表示数字是
    单精度浮点类型的

C语言允许程序员扩展新的数据类型
这些数据类型叫复合数据类型
复合数据类型需要先声明然后才能使用

布尔类型是C99规范中引入的新的数据
    类型
布尔类型里包含两个整数，分别是0和1。
    0叫做假，1叫做真

任何整数都可以当作布尔值使用，整数
    0当布尔值使用的时候是假，其他所有
	整数当布尔值使用的时候都是真

数据类型和占位符的对应关系
char和unsigned char       %c
short                     %hd
unsigned short            %hu
long                      %ld
unsigned long             %lu
int                       %d
unsigned int              %u
float                     %f或%g
double                    %lf或%lg
     %f和%lf会保留小数点后无效的0
     %g和%lg不会保留	 

不同类型存储区中包含的字节个数不同
sizeof关键字可以用来计算一个存储区
    或数据类型中包含的字节个数

char和unsigned char        1个字节
short和unsigned short      2个字节
long和unsigned long        4个字节
int和unsigned int          4个字节
float                      4个字节
double                     8个字节

sizeof关键字后面的小括号里可以写
    任何能当作数字使用的内容
sizeof小括号里修改了任何存储区的内容
     都不会真正发生

scanf标准函数可以从键盘得到用户
    输入的内容并记录在存储区里
为了使用这个标准函数也需要包含
    stdio.h头文件
scanf函数调用语句里要使用存储区
    地址表示存储区
scanf函数调用语句的双引号里要使用
    占位符表示存储区的类型
scanf函数调用语句双引号里不要写不是
    占位符的内容
如果用户输入的数字格式和程序要求
    的格式不一致就有可能导致程序
	无法得到数字

一个字节可以分成八段，每段可以记录
    一个0或者1
要想把一个数字记录到一个字节里必须
   首先把它拆分成八个0或者1
拆分后的数字表示方式叫做二进制
任何数字既可以用十进制方式表示也
    可以用二进制方式表示
二进制表示方式中每个数位有一个编号，
    最右边数位的编号是0向左依次
	递增
如果某个数位上是1则它代表2的数位
    编号次方
如果两个相邻数位上数字一样则它们
    之间有倍数关系
使用二进制表示的非负数符合以上规则
二进制加一的计算规则是把最右边的
    所有连续的1都变成0，把最右边
	的0变成1

二进制表示的非负数转换成十进制的时候
    只要把每个数位上的数字单独转换
	然后求和就可以了

0000 0101 = 2的0次方 + 2的2次方
          = 1 + 4
		  = 5

十进制表示的非负数转换成二进制的方法
*******0          12
******0            6
*****1             3
****1              1
****               0

00001100

把数字不停除以2保留整数部分得到一组
    数字，用每个数字除以2取余得到
	一组0或者1。把这些0或者1按照
	倒序书写就得到转换结果。
